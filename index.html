Hospedagem GitHub

Para criar um site no GitHub.com devemos ter uma conta válida e um repositório com um arquivo index.html

Hospedando um site

  1. Criar um repositório;
  2. Publicar o repositório utilizando o publish;
  3. Abrir o repositório no VSCode;
  4. Criar o index.html;
  5. Criar a versão usando commit;
  6. Enviar para o repositório remoto usando push;
  7. Entrar no GitHub.com, acessar o repositório --> settings --> pages --> escolher a branch do source;
  8. Clicar na URL gerada.
